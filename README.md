## MatPFF
MatPFF: Matlab FEM program for phase-field fracture (AT1/2, PFCZM) simulation

## Features

- Two-dimensional linear elastic phase-field fracture 
- AT1 & AT2  phase-field models, PFCZM is added in 24-12-12
